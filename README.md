Dba Automation Sample
Database Automation & Monitoring Portfolio Sample (NDA-Compliant)

Overview

This repository contains NDA-compliant examples of database automation, monitoring, and maintenance workflows for PostgreSQL environments. It demonstrates practical skills in automating backups, monitoring performance, and implementing alerting mechanisms while ensuring data security.

Features

Automated Backup Script: Full logical backup using pg_dump, compressed and optionally uploaded to cloud storage.

Monitoring Snippets: Queries for active connections, slow queries, and replication lag.

Alerting Logic: Threshold-based alerts for CPU, connections, replication lag, and slow queries.

NDA-Compliant: Fully anonymized, no sensitive or proprietary data.
